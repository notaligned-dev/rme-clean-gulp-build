# Журнал изменений
Все значимые изменения проекта будут документироваться в этом файле.

Формат основан на [Keep a Changelog](https://keepachangelog.com/ru/1.1.0/),
а проект следует принципам [Семантического версионирования](https://semver.org/lang/ru/).

## [Новое]

### Добавлено

- `gulp-if` модуль (условные операторы для пайплайнов)
- новая опция в конфигурации: включение/выкчючение конверсии в `webp`

### Исправлено

- Выходной путь JS библиотек

## [0.2-beta.0] — 2026-02-10

### Добавлено

- Установлен пакет `cross-env` для кроссплатформенных скриптов

### Изменено

- `config.js` заменён на новую систему конфигурации (**Критическое изменение**)
- Новое расположение сабмодуля темы RME: `src/assets/theme` → `src/rem-markup-engine-theme`
- Исходники-примеры заменены на `.gitkeep` для сохранения структуры папок
- Точка входа Gulp (`gulpfile.js` теперь файл, и директория gulp переименована `gulp`)
- Переменные путей для исходных файлов и `browser-sync`

### Исправлено

- Задача для отслеживания изменений скриптов

### Удалено

- Неиспользованные переменнеые в `gulp/globals/paths.js`

## [0.1-beta.1] — 2026-02-05

### Исправлено

- Исправлены пути для задач Gulp (были сломаны)

## [0.1-beta.0] — 2026-02-04

### Добавлено

- Первый публичный релиз
- Сборка Gulp для SCSS, JavaScript и ассетов
- Интеграция ESBuild для сборки JavaScript
- Компиляция SCSS с sourcemaps и оптимизацией для production
- Поддержка BrowserSync с автоматической перезагрузкой
- Опциональный сабмодуль Rem Markup Engine Theme
- Конфигурации линтеров для JS
